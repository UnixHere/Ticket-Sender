# 🎟 Ticket Verifier
A locally-hosted mobile-friendly web app that scans QR codes from your tickets and checks them against `students_database.xlsx`.
---
## Features
- 📷 **QR Scanner** — uses the phone camera to scan ticket QR codes
- ✅ **Instant verification** — checks the QR code against your Excel database
- ⚠️ **Duplicate detection** — warns if a ticket has already been scanned
- ✋ **Check-in button** — marks a student as arrived directly in the Excel file (column F)
- 🔍 **Manual lookup** — search by name, ID, class or email
- 📋 **Attendees list** — full list with live arrived/pending stats
- ↩ **Undo** — un-check someone if needed
---
## Setup
### 1. Install dependencies
```bash
pip install flask flask-cors openpyxl
```
Or with uv:
```bash
uv add flask flask-cors openpyxl
```
### 2. Copy your database
Put your `students_database.xlsx` in the same folder as `app.py`.
The app expects the same column layout as Ticket-Sender:
| A — Name | B — Class | C — ID | D — Email | E — Sent | F — Arrived (added by this app) |
Column F (`Arrived`) is created automatically on first run.
### 3. Run
```bash
python app.py
```
The terminal will print your local IP address, e.g.:
```
  → http://192.168.1.42:5000
```
### 4. Open on phone
Connect your phone to the **same WiFi** and open that URL in the browser.
> On iPhone: Safari works best for camera access.
> On Android: Chrome works best.
---
## How QR codes are read
The QR codes generated by Ticket-Sender encode:
```
Name | Class | ID
```
e.g. `Jana Nováková | 4.A | 142`
The verifier parses this and looks up the ID in your database.

## Excel columns
| Col | Header   | Used by          |
|-----|----------|------------------|
| A   | Name     | Both             |
| B   | Class    | Both             |
| C   | ID       | Both             |
| D   | Email    | Ticket-Sender    |
| E   | Sent     | Ticket-Sender    |
| F   | Arrived  | **Ticket Verifier** (added automatically) |
